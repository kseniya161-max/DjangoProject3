# Название проекта
DjangoProject3

# Описание проекта
1.В приложении каталога new_app созданы модели Product и Category с базовыми настройками.
Также проект настроен для работы с базами данных.


## Установка

1. **Клонируйте репозиторий:**
   https://github.com/kseniya161-max/DjangoProject3/pull/1
2. Установите Poetry (если он еще не установлен)

3. Установите зависимости:
   poetry install

4. Запустите проект в виртуальном окружении Poetry:
   poetry shell
## Запуск проекта
   1. Скопируйте файл .env.sample и переименуйте  его в .env.
   2. Заполните .env своими данными.
   3. Убедитесь, что установлены все зависимости
   4. Запустите проект: poetry run python manage.py runserver
